Google Fonts

  1. Open Google Fonts in your web browser
  2. In the search bar, type your desired font. For example, type Poppins
  3. Click on the font & then select Get Font
  4. Select Get Embed Code
  5. Select Import
  6. Copy out the code in the box under Embed code in the <head> of your html

    It should like something like this: 
    
<style>
@import url('https://fonts.googleapis.com/css2?
family=Poppins:ital,wght@0,100;0,200;0,300;0,400;
0,500;0,600;0,700;0,800;0,900;1,100;1,200;1,300;
1,400;1,500;1,600;1,700;1,800;1,900&display=swap');
</style>
  7. Paste this code in between the <head></head> tags of your HTML file

    See example below: 
    
<!DOCTYPE html>
<html lang="en" 
xmlns="http://www.w3.org/1999/xhtml" 
xmlns:v="urn:schemas-microsoft-com:vml" 
xmlns:o="urn:schemas-microsoft-com:office:office">
<head>
<link rel="SHORTCUT ICON" href="/e/ref.ico" />
<meta name="robots" content="noindex, nofollow" />
<meta name="ROBOTS" content="NOINDEX, NOFOLLOW" />
<!-- Check preferred mode - light or dark -->
<meta name="color-scheme" content="light dark" />
<meta name="supported-color-schemes" content="light dark" />

<meta name="viewport" content="width=device-width, 
initial-scale=1.0, maximum-scale=1.0, user-scalable=no">
<title>TITLE HERE</title>
<!-- ADD GOOGLE FONTS HERE -->	
<!-- Add google font to start of font family i.e. 
font-family: Roboto, Arial, Helvetica, sans-serif; -->
<style>
@import url('https://fonts.googleapis.com/css2?
family=Poppins:ital,wght@0,100;0,200;0,300;0,400;0,500;
0,600;0,700;0,800;0,900;1,100;1,200;1,300;1,400;1,500;
1,600;1,700;1,800;1,900&display=swap');
</style>
<!-- END ADD GOOGLE FONTS HERE -->
  8. You have now linked the Google Font Family to your HTML
  9. To apply the Font-Family to your text, add the font family name to the front of your font-family code like this: 
    font-family: 'Poppins', Arial, Helvetica, sans-serif;
